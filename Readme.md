A comprehensive, role-based online examination platform built with modern Java technologies. This system streamlines the entire exam lifecycle, from **question bank management and intelligent paper generation to online proctoring, automated grading, and performance analytics**.
## 1. 智能组卷与防作弊机制

### 需求描述
系统需支持基于知识点、难度系数、题型分布的智能自动组卷功能，同时集成多维度防作弊策略。

### 详细要求
- **智能组卷**：教师可设定各题型数量、分值、难度比例，系统从题库中随机抽取符合规则的题目生成试卷，支持手动调整与预览。
- **防作弊策略**：
  - 随机打乱题目顺序及选项顺序。
  - 限制切屏次数（如超过3次自动交卷）。
  - 摄像头定时抓拍与异常行为检测（如多张人脸、离座）。
  - 试题不可回溯（答完一题后禁止返回修改）。
- **考试异常恢复**：因网络中断等原因退出后，再次进入可恢复答题进度（保存已答记录）。

## 2. 多角色工作台与成绩分析

### 需求描述
平台需为管理员、教师、学生分别提供独立的工作台，并支持多维度的成绩统计分析及导出。

### 详细要求
- **角色工作台**：
  - **学生**：查看待考/历史考试、成绩、错题本、排名。
  - **教师**：创建/发布考试、监考视图（实时查看考生状态）、批阅主观题、发布成绩。
  - **管理员**：用户管理、角色权限分配、系统日志审计。
- **成绩分析**：
  - 自动生成班级/年级的平均分、最高/低分、及格率、分数段分布图。
  - 每道题的答对率、常见错误选项统计。
  - 支持导出为 Excel/PDF 报告，并支持对比不同班级或历次考试趋势。
- **错题本**：系统自动收集学生错题，按知识点归类，支持推送同类练习题。
